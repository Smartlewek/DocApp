## Instalation
* Script

## Script
To run this project, install it locally using setup bash script.
Clone setup script:
```
$ wget http://raw.githubusercontent.com/Smartlewek/DocApp/main/setup.sh
```
Add Permision
```
$ chmod +x setup.sh
```
Run Script 
```
$ ./setup.sh
```
